# Array Processing Program – MIPS Assembly

A MIPS Assembly program developed using the MARS simulator to perform array processing, classification, sorting, and statistical analysis entirely at the assembly level. The program reads user input, separates even and odd integers into dedicated arrays, combines them into a single array, and sorts the resulting data in ascending order while operating directly on registers and memory.

The implementation follows standard MIPS calling conventions, utilizing subroutines, argument passing, return values, and stack management. Additional functionality includes counting positive and negative numbers, calculating the array average, and displaying sorted results through syscall-based input/output operations. The project demonstrates practical experience with low-level programming, memory manipulation, ISA-level I/O, and the hardware-software interface.
